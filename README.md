# Prateek Kr. Singh Portfolio

This repository contains a single-page professional portfolio for **Prateek Kr. Singh**.

## Free GitHub URL (GitHub Pages)

After pushing this repository to GitHub, enable **GitHub Pages**:

1. Go to **Repository → Settings → Pages**.
2. In **Build and deployment**, select:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (or your default branch), folder `/ (root)`
3. Save.

Your free portfolio URL will be:

`https://<your-github-username>.github.io/<repository-name>/`

Example (if repo is `portfolio`):

`https://<your-github-username>.github.io/portfolio/`

## Local Preview

Open `index.html` directly in your browser.
